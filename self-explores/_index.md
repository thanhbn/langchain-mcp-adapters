# Project Context Index

Folder này lưu trữ learnings, decisions, và context quan trọng của dự án **langchain-mcp-adapters**.
Claude sessions sau sẽ đọc folder này để hiểu bối cảnh.

## Cấu trúc
- `decisions/` — Quyết định quan trọng
- `learnings/` — Kiến thức học được
- `context/` — Bối cảnh dự án
- `daily/` — Nhật ký hàng ngày
- `tasks/` — Context cho từng task (worklog)
- `leverage/` — Phân tích đòn bẩy
- `samples/` — Mẫu công việc tái sử dụng
- `history/` — Lịch sử tương tác /viec (JSONL)

## Cách sử dụng
- `/viec ghi` — Ghi chép mới (learning, decision, context)
- `/viec doc` — Đọc lại context đã ghi
- `/viec bat-dau` — Nhận task, tự động tạo worklog
- `/viec xong` — Tự động capture output khi hoàn thành task
- `/viec review` — Tổng hợp vào daily log
